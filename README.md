# AI-ML-Based-Islamabad-Housing-Price-Prediction-System
# 🏡 AI & ML-Based Islamabad Housing Price Prediction System

## 📖 Overview

This project is a Machine Learning and AI-powered web application developed as a Final Year Project for a Computer Science degree. The system predicts housing prices in Islamabad based on user-provided inputs such as location, area (in Marla), number of bedrooms, and bathrooms.

The goal of this project is to provide a smart, data-driven solution for real estate price estimation, helping users make informed decisions.

---

## 🚀 Features

* 🔍 Predict house prices using Machine Learning algorithms
* 🌐 Web-based interface built with Flask
* 📊 Data cleaning and preprocessing using Pandas
* ⚡ Fast and real-time predictions
* 🧠 AI-driven decision-making system
* 🎯 User-friendly UI using HTML, CSS, and Bootstrap

## ⚙️ How the System Works

1. **Data Collection**
   Real estate data of houses in Islamabad is collected.

2. **Data Cleaning & Processing**

   * Handled missing values
   * Converted categorical data
   * Feature selection using Pandas

3. **Model Training**

   * Machine Learning algorithm trained on housing data
   * Model saved using Pickle

4. **Web Integration**

   * Flask is used to connect the ML model with the frontend
   * User inputs are sent to backend

5. **Prediction**

   * Model processes inputs
   * Predicted price is displayed instantly
